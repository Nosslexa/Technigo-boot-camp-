API

Application Programming Interface

What does an API do?

It's an application intermediary that allows two applications, computers or programs to talk to each other.
API´s can also control access to hardware devices or software functions. That's why API often plays a role in security.

How does it work?

Imagine you are booking a flight on the internet at a site where you can compare different flights and prices.
API´s is what makes the site able to gather information about prices, dates and different departure and arrival times from the different airlines.

Another example to explain an API is to compare it to a restaurant.
You have a menu to order from. The Kitchen is the “system” and will prepare your order.
What you need now is the ability to communicate your order to the kitchen.
This is where the API comes in, aka the waiter.
The waiter takes your order/request to the kitchen who now can prepare your order. Then the waiter delivers the food back to you.

So in this example the API communicates your request to the system and then interprets that data and presents you with the information you wanted.
